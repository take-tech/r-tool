# R-Tool

R-Tool は、ゲイン調整・L/R クロスフェード・パン・ステレオ幅調整・アウトプットメーターをひとつにまとめた、軽量なステレオユーティリティプラグインです。

ミックス作業で頻繁に必要になる処理をまとめ、どのトラックにも気軽に挿して使えることを目指しています。

---

## 機能

| コントロール | 概要 |
|---|---|
| **Volume** | 音量調整（-∞ ～ +18.1 dB） |
| **Gain** | ワイドレンジのゲイン調整（-36 ～ +36 dB） |
| **Pan** | イコールパワー・ステレオパン（L100 ～ C ～ R100） |
| **Width** | M/S方式によるステレオ幅調整（0% ～ 200%） |
| **Swap** | L/R クロスフェード（0% = そのまま / 50% = モノラル / 100% = 左右入れ替え） |
| **Output Meter** | 全処理後の L/R ピークメーター（目盛り付き） |

### Swap ノブについて

オートメーションを使えば、ステレオトラック上でチャンネルをなめらかに移動させることができます。たとえば 2 トラックのモノ素材を 1 本のステレオトラックにまとめ、Swap を書き込むことで左右の割り当てを動的に切り替えられます。

## ビルド方法

### 必要な環境

- CMake 3.22 以上
- C++17 対応コンパイラ（Xcode / MSVC）
- サブモジュールの初期化済みであること

```bash
git submodule update --init --recursive
```

### 手順

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

ビルド成果物は `build/RTool_artefacts/` 以下に出力されます。
